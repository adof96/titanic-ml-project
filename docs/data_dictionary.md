# Data Dictionary - Titanic Dataset

| Column | Type | Description |
|---|---|---|
| PassengerId | int | Unique passenger identifier |
| Survived | int | Survival status (0 = No, 1 = Yes) |
| Pclass | int | Passenger class |
| Name | object | Passenger name |
| Sex | object | Passenger gender |
| Age | float | Passenger age |
| SibSp | int | Number of siblings/spouses aboard |
| Parch | int | Number of parents/children aboard |
| Fare | float | Ticket fare |
| Embarked | object | Port of embarkation |
